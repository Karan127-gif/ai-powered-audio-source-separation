How the App Works — AI Pipeline

🧠 What Does Audio Separation Mean?

The app takes a mixed music file (like a full song with vocals, drums, bass and instruments all together) and uses a trained AI model to split it into separate tracks — one for each instrument. This is like unmixing a bowl of soup back into its ingredients.

Step 1 — Load Audio

The uploaded MP3/WAV file is loaded and resampled to 22,050 Hz (mono-compatible stereo). Handles both mono and stereo files.

Step 2 — STFT (Spectrogram)

Audio is converted from a waveform (time domain) to a spectrogram (frequency vs time) using Short-Time Fourier Transform. This is what the AI model reads — like turning sound into a picture.

Step 3 — HPSS (Harmonic-Percussive Separation)

Before the AI runs, we separate harmonic content (vocals, melody) from percussive content (drums, hits). This helps the AI produce cleaner stem masks.

Step 4 — UNet Model Inference (Chunked)

The AI model (MultiStem UNet) processes the spectrogram in overlapping 4-second chunks. It was trained on 300 epochs of music data to predict 4 masks — one for each stem: Vocals, Drums, Bass, Other.

Step 5 — 2-Pass Wiener Filter

A classical signal processing technique that sharpens the AI's masks. Run twice — each pass uses the previous output as a refined prior, producing much cleaner separation.

Step 6 — Griffin-Lim Phase Reconstruction

After masking, the vocals stem is reconstructed using Griffin-Lim (iterative phase estimation). Without this, audio sounds "watery" or "metallic".

Step 7 — Post-Processing & Save

Each stem gets per-stem spectral cleanup (bandpass filters, noise gates), loudness normalisation, and hard-limiting to prevent clipping. Final WAV files are saved to the separated_output/ folder.

Troubleshooting

⚠️ "Windows protected your PC" warning appears when opening the EXE

This is a Windows SmartScreen warning for unsigned EXEs. The file is safe — it was built from the included launcher.py source code. To bypass:

  1. Click "More info" in the warning dialog.
  2. Click "Run anyway".
  3. The single setup window will open and start working automatically.
⚠️ Setup window shows a red "Failed to install PyTorch" error

The error is shown inside the setup window in red — no extra popup. This is usually an internet connection issue. Try:

  1. Make sure you are connected to the internet (not just a local network).
  2. Disable your VPN if you have one.
  3. Close the setup window and double-click Launch AI Audio Separator Pro.exe again — it will retry all steps from the beginning.
  4. Alternatively, use Run App.bat — it shows more detailed error messages in a terminal window.
⚠️ App launches but model fails to load

The file multistem_unet_best.pth must be in the same folder as main.py. Make sure you didn't move individual files out of the extracted folder. Always keep all files together.

⚠️ Separation takes very long

The app runs on CPU only (no GPU). Processing time depends on your PC's speed and the length of the audio file. A 3-minute song typically takes 2–5 minutes on a modern CPU. The included test_song.wav is about 42 seconds and processes much faster.

⚠️ "Python is not installed" error in Run App.bat
  1. Download Python from https://www.python.org/downloads/
  2. During installation, check the box: "Add Python to PATH" — this is critical.
  3. After installation, close and reopen the .bat file.
⚠️ No sound / output WAV files are silent

Make sure the input audio file is not already a stem (e.g. don't separate an already-separated vocals track). Use a full mixed song. The test_song.wav included is a good starting point to verify everything is working.
